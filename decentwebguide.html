Decent Website Layout Guide

Display & Position

Attribute What It Does
Position: Static Default: element stays in its regular position
Position: Relative The element is placed in its regular position, then top, right, bottom, and left is used (given with a PX value, e.g. top: 5px;) to adjust the position.
Position: Absolute The element is removed from the normal document flow, no space is created for the element, and it is positioned relative to its closest positioned ancestor or containing block.
Position: Fixed The element is removed from the normal document flow, like absolute, but stays in place as the user scrolls.
Display: Block The element generates a block box, generating line breaks both before and after the element when in the normal flow.
Display: Inline The element does not make a new line, cannot be sized using height/width properties.
Display: Inline-Block The element does not make a new line, but can be size using height/width properties.
Display: Flex The element behaves like a block-level element and lays out its content according to the flexbox model.
Display: Grid The element behaves like a block-level element and lays out its content according to the grid model.

Flexbox

Attribute What It Does
Position: Static Default: element stays in its regular position
Position: Relative The element is placed in its regular position, then top, right, bottom, and left is used (given with a PX value, e.g. top: 5px;) to adjust the position.

Flexbox

Attribute What It Does
align-content Sets the distribution of space along the element's cross axis
align-items controls the alignment of items on the cross axis.
align-self Overrides the above settings, for a specific element
flex-basis Sets the initial main size of a flex item
flex-direction The direction for items to flow (row/column - default row)
flex-flow Shorthand for direction and wrap. Example: flex-flow: column wrap;
flex-grow Sets how much the element should grow relative to other items in the flexbox
flex-shrink Sets how much the item should shrink relative to other items in the flexbox.
flex-wrap Sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.
justify-content Defines how space is distributed along the main axis (e.g. horizontal, if direction is set to row).
